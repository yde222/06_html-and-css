선택자[속성=속성값] {설정내용}의 형식으로 사용된다.

동위선택자

동위 관계(형제 관계)에서 뒤에 위치한 태그를 선택할 때 사용한다.

바로 뒤의 요소 선택 시: 선택자 a+ 선택자 b{설정내용} 으로 사용하며,
        뒤에 있는 모두를 선택할 때는 선택자 a~ 선택자 b{설정내용}으로 사용된다.

상태 선택자

입력 양식의 상태에 따라 선택되는 선택자

체크 상태의 input 태그 선택: input태그 : checked {설정내용}

초점이 맞추어진 input태그 선택



사용 가능한 input 태그 선택

태그:enabled{설정내용}, 태그 : disabled{설정내용}
